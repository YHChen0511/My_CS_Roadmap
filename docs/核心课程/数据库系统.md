# 数据库系统

数据模型（关系型、文档型、键值型）、存储模型（N-元、分解）、查询语言（SQL、存储过程）、存储架构（堆存储、日志结构存储）、索引（顺序保持树、哈希表）、事务处理（ACID、并发控制）、恢复机制（日志、检查点）、查询处理（连接、排序、聚合、优化）以及并行架构（多核、分布式）
